history of HTML:

        1. **1989-1991:** Tim Berners-Lee invents HTML to share documents at CERN.
        2. **1995:** HTML 2.0 introduces forms and tables.
        3. **1997:** HTML 3.0 adds more features but faces compatibility issues.
        4. **1999:** HTML 4.01 brings CSS, JavaScript, and multimedia support.
        5. **2000:** XHTML attempts stricter syntax using XML but isn't widely adopted.
        6. **2014:** HTML5 introduces semantic elements, audio/video support, canvas, and offline apps.
        7. **Ongoing:** HTML evolves as a "living standard," continuously updated to adapt to technology changes.
